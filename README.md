# Домашнее задание к работе 12 #
## Условие задачи ##
Напишите программу, которая вычисляет размер основных типов данных в байтах: char
## 1. Алгоритм блок-схема ##
### Алгоритм ###
1. Начало
2. Выводим текст с указанием заданного типа
3. Выводим количество байт для данного типа на экран: ```sizeof(char)```
4. Конец
### Блок-схема ###
![](https://github.com/kiri4art/Lab12/blob/main/lab12alg.png)
## 2. Реализация программы ##
```
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <locale.h>
#include <stdlib.h>

int main() 
{
    system("chcp 1251");
    printf("\nТип данных: char\n");
    printf("Размер данных данного типа = %d байт\n", sizeof(char));
    return 0;
}
```
